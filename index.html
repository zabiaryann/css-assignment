Section 1: Selectors in CSS Demo

Thanks for visiting my aesthetic page. This is an example of how to use semantic HTML. Remember, semantic HTML refers to the practice of using HTML elements to convey the meaning or structure of the content on a web page, rather than just using generic containers like div or span. Notice how this page has, a header, a nav, a main, and multiple section elements. These don't affect how your website looks, but they better organize your site and also help increase your site's accessibility for people that use screenreaders to browse the web
